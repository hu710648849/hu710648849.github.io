hexo+github+metary博客搭建总结

html css javascript hexo+github
Publish Date:   2022-06-23

使用Github+hexo搭建个人博客(hututu的第一篇博客噢,项目请看这里https://github.com/hu710648849/hu710648849.github.io.git)

1.第一条hututu博客的说明:

​ 首先想说的是本人很高兴经过两年的学习，对于自己做博客还是有兴趣去做并且尝试着去设计使用，这个hexo+github，主题为metary的搭建是我花费了暑假这三天时间整出来的一个玩意，主要是效仿各个资料csdn、百度各个地方等，刚开始是致力于准备随便搭建一个可以在网站上通过网址可以查看的自己的博客，用来记录自己的学习过程和遇到的问题解决，让自己的思维变得更加的严谨，培养一下做事的耐性，因为记录博客需要花费大量的精力和时间，希望这个博客可以就此慢慢展开，后来的一切都可以慢慢变好

2.搭建blog所需要的基本环境配置和准备操作（作者所使用系统为windows,如果有mas，linux等其他系统，小白可以自行参考其他网站噢，但是大体差不多）

  1. 安装好node.js环境 git环境（两者都要配置好相关的环境变量，便于在命令行中敲相关命令）

  2. 可在cmd中查看安装版本，即检验是否成功能使用（node -v ; git -version）

  3. 因为你身需要的是在网址能使用查看（我的初衷），当然满足手机和客户端，github是个不错的选择，所以注册一个github账号（略）

  4. 在github上建立一个属于你自己的仓库（查看相关文件说是仓库名字和用户名保持一样的，在github page文档中可查看详情https://docs.github.com/cn）

  5. hexo官方文档

  6. 参考博客

3.本地配置安装hexo

  1. 任意位置创建一个blog文件夹

  2. 当时我的在安装hexo时候总报一个错所以直接指定淘宝镜像安装的npm install -g hexo- cli --registry=https://registry.npm.taobao.org

  3. 正式开始安装hexo命令 npm install -g hexo-cli

  4. 初始化hexo init

  5. hexo s 运行打开的博客在locahost:4000中（本地查看一般用于测试，hexo clean hexo g hexo s hexo d）

  6. 打开默认为landscape主题theme我的文件中还有两个主题这个不同主题可以自己去某度上搜索

4.将本地文件上传到github上（后面可以通过访问<www.hu710648849.github.io.git>来查看你的博客）

​ 1.输入cd ~/.ssh进入.ssh文件 (这里也可以使用作者没用)

​ 2.ssh-keygen -t rsa -C "710648849@710648849.com" （直接使用我是 (一直摁enter ,yes 就敲yes)）

​ 3.出现了一个rsa乱码后你的c盘下面会生成一个id_rsa 和 一个id_rsa.pub文件

​ 4.打开记事本复制这个id_rsa.pua的内容

​ 5.在github头像下面的setting里面的SHA and GPG keys中复制添加上述全部内容点击add SSH key

​ 6.本地进行验证一次 输入命令：ssh -T git@github.com 并且输入yes之后，行末尾会显示你的用 户名(好像可以忽略)

​ 7.绑定邮箱和用户名（git config --global user.name “hu710648849” git config --global user.email “710648849@qq.com”)

​ 8.作者是没有验证查看第六步 直接可以blog目录下使用绑定

​ 9.上传博客修改本地仓库目录下_config.yml文件 注意后面空格

​ type: git

​ repository: ·····

​ branch: master

​ 10.安装上传工具npm install hexo-deployer-git

​ 11.后续编写可以直接hexo d 进行博客编写

5.浏览器输入：用户名.github.io(Github仓库名)就可以访问博客啦
